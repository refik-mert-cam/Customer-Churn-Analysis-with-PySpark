# Customer Churn Analysis with PySpark

This repo demonstrates an end-to-end PySpark workflow on a synthetic customer churn dataset:
- Generate a dataset (no external downloads)
- Explore schema + basic stats
- Feature engineering
- Churn metrics / segment analysis
- Optional ML baseline with Spark ML (Logistic Regression)
